# To Do list App

Aplikacja do zarzadzania lista todo projektów

## Podstawowe funkcjonalnosci

-   Logowanie
-   Prywatna lista zadań
-   Grupowa lista zadań
-   Zapraszanie do grupy
-   Tworzenie linku z zaproszeniem
-   Zarządzanie członkami grupy
